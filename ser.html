📘 Для русскоговорящих • A1 • Начинающие

📖 ¿Qué es el verbo SER? (что значит "ser")

Ser significa "быть" en español. Se usa para identificar, describir características permanentes, origen, profesión, nacionalidad y hora. 🧠 ¡Es muy importante!

📌 CONJUGACIÓN (спряжение) 👇

PronombreVerbo SERTraducción (русский)
Yosoyя есть / я являюсь
Túeresты есть
Él / Ella / Ustedesон/она есть
Nosotros/assomosмы есть
Vosotros/assoisвы есть (мн. ч., Испания)
Ellos / Ellas / Ustedessonони есть

💡 En América Latina se usa "ustedes" → son (vosotros se usa poco). ¡No te preocupes! ✨

📝 EJEMPLOS BÁSICOS (con emojis)

🇪🇸 Yo soy estudiante. → 📚 Я студент.
🇪🇸 Tú eres inteligente. → 🧠 Ты умный.
🇪🇸 Él es médico. → 🩺 Он врач.
🇪🇸 Nosotros somos amigos. → 👫 Мы друзья.
🇪🇸 Ellas son españolas. → 🇪🇸👩她们 испанки.
🇪🇸 Hoy es lunes. → 📅 Сегодня понедельник.
🇪🇸 Mi casa es grande. → 🏠 Мой дом большой.
⚡ Regla rápida: Ser = características permanentes / identidad / origen. (В отличие от estar — временное состояние).

🧩 ¡Practiquemos! / Практикуем! 🧩

📌 Opción múltiple (выбери правильную форму)

1️⃣ Yo _______ estudiante de español. 🤓

2️⃣ Ana y María _______ inteligentes. 🧠✨

3️⃣ Nosotros _______ de Rusia. 🇷🇺❄️

4️⃣ ¿Tú _______ profesor? 👨‍🏫

✔️❌ Verdadero o Falso (правда или ложь?)

1. "Nosotros somos" significa "мы есть" — это правильно. 🤝

2. "Ella es médico" → incorrecto, debería ser "Ella soy médico". 🩺

3. "Yo eres de Madrid" es la forma correcta del verbo SER. 🏙️

4. "Ustedes son amables" (вы вежливы) — conjugación correcta. 😊

🌟 ¡Recuerda! SER = identidad, origen, profesión, características permanentes.
